# NuShell Community

![nushell](/images/nushell.png)

## What is it?

NuShell Community is a plugin to highlight syntax nu scripts.

## How to use it?

- Add plugin to Intellij IDE
- Open a nu script

### Add plugin to Intellij IDE

- Install from plugin store

  ![Market Place](/images/marketplace.png)

- Plugin url

  [Plugin Download](https://plugins.jetbrains.com/plugin/22953-nushell-community)
